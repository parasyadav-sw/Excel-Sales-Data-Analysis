# Sales Data Analysis Using Excel

## Project Overview
This project focuses on analyzing real-world sales data using Microsoft Excel.  
The dataset was cleaned, transformed, and analyzed to generate meaningful business insights through Pivot Tables and interactive Dashboards.

---

## Skills & Features Used
- Data Cleaning
- Data Transformation
- Remove Duplicates
- Filters
- Data Type Formatting
- Excel Formulas
- VLOOKUP
- Pivot Tables
- Dashboards
- Slicers
- Data Visualization

---

## Work Performed

### Data Cleaning & Preparation
- Converted raw data into an understandable structured table
- Removed duplicate records
- Applied filters for easier analysis
- Transformed and corrected data types

### Data Analysis
- Used Excel formulas for calculations
- Applied VLOOKUP for data mapping and retrieval
- Created Pivot Tables for summarizing data

### Dashboard Creation
- Built an interactive dashboard
- Added slicers for dynamic filtering
- Created charts and visual reports for business insights

---

## Key Insights
- Identified top-performing categories and regions
- Analyzed sales and profit trends
- Compared performance across different segments
- Generated interactive business reports
